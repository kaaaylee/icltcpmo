Getting Started

  1. Clone the repo: git clone https://github.com/kaaaylee/icltcpmo
  2. Install textX: pip install textX
  3. Run a program: python interpreter.py program.pmo

About

icl tc pmo (icl = i cant lie tc = this code pmo = piss me off) is a custom programming language built using textX, made for my advanced programming languages class. It's an esoteric language inspired by LOLCODE, a popular esoteric language that resembles the speech of lolcats (old internet meme), and is meant to be humorous and playful. The language uses keywords inspired by common online slang and acronyms like "wsg," "ong," "sybau," and "icl ts pmo."
